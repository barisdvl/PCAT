## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/barisdvl/PCAT.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start project
   ```js
   npm start
   ```
